BloodTrack 日次記録 (2026 start)
Files:
- index.html
- style.css
- script.js
- manifest.json
- sw.js

Usage:
1. Extract the zip and open index.html in Chrome/Edge.
2. Input data by date. Use '下書き保存' to temporarily store an in-progress entry.
3. '保存' will store the entry permanently (localStorage). You can edit entries from the table.
4. Use 月次レポート to view monthly averages and chart.
5. Export CSV for backups.
Note: Data is stored locally in your browser's localStorage.
