BloodTrack 日次記録 v2 (印刷/PDF対応)
Files:
- index.html
- style.css
- script.js
- script_print.js
- manifest.json
- sw.js

Usage:
1. Extract the zip and open index.html in Chrome/Edge.
2. Input daily data and save.
3. In '印刷用 月次表' select a month, click '表を表示' to render the month in a table.
4. Click 'PDFで出力' to download an A4-landscape PDF named '血糖値記録_YYYY-MM.pdf'.
Note: Data is stored locally in your browser's localStorage.
